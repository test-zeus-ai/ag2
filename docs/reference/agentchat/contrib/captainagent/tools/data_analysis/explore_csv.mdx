### explore\_csv

```python
def explore_csv(file_path, num_lines=5)
```

Reads a CSV file and prints the column names, shape, data types, and the first few lines of data.

**Arguments**:

- `file_path` _str_ - The path to the CSV file.
- `num_lines` _int, optional_ - The number of lines to print. Defaults to 5.
