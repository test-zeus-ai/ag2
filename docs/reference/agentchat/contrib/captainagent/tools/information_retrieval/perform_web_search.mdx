### perform\_web\_search

```python
def perform_web_search(query, count=10, offset=0)
```

Perform a web search using Bing API.

**Arguments**:

- `query` _str_ - The search query.
- `count` _int, optional_ - Number of search results to retrieve. Defaults to 10.
- `offset` _int, optional_ - Offset of the first search result. Defaults to 0.
  

**Returns**:

  The name, URL and snippet of each search result.
