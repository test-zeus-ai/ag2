### with\_requirements

```python
def with_requirements(
    python_packages: list[str] = [],
    global_imports: list[Import] = []
) -> Callable[[Callable[P, T]], FunctionWithRequirements[T, P]]
```

Decorate a function with package and import requirements

**Arguments**:

- `python_packages` _List[str], optional_ - Packages required to function. Can include version info.. Defaults to [].
- `global_imports` _List[Import], optional_ - Required imports. Defaults to [].
  

**Returns**:

  Callable[[Callable[P, T]], FunctionWithRequirements[T, P]]: The decorated function
