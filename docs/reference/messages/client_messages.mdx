## ModelUsageSummary

```python
class ModelUsageSummary(BaseModel)
```

Model usage summary.

#### model

Model name.

#### completion\_tokens

Number of tokens used for completion.

#### cost

Cost of the completion.

#### prompt\_tokens

Number of tokens used for prompt.

#### total\_tokens

Total number of tokens used.
